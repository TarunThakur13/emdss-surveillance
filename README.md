# Emergency Motion Detection Surveillance System

This project is a web-based surveillance application
developed using Flask and OpenCV.

It streams live video from a camera, detects motion
in real time, and displays the feed on a web dashboard.
Users can toggle motion detection and capture snapshots.

Technologies used:
- Python
- Flask
- OpenCV
- HTML, CSS, JavaScript

Future Scope:
- Email alerts on motion detection
- Multiple camera support
- Cloud deployment
